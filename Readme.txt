== Description ==

A simple WordPress plugin to add a customizable topbar to your site. This plugin allows you to easily configure the appearance and content of the topbar through the WordPress admin panel. 

Features:
- Choose between a fixed or scrollable topbar.
- Customize background color and text color with color pickers and hex codes.
- Select font from available fonts in WordPress.
- Set font size and height with units like px, em, rem, etc.
- Add custom text content to the topbar.
- Include additional custom CSS for further customization.
- Easily manage settings through the WordPress admin panel.

== Installation ==

1. Upload the `custom-topbar` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' > 'Custom Topbar' to configure the plugin settings.

== Frequently Asked Questions ==

= How do I customize the topbar's appearance? =

You can customize the topbar's appearance through the plugin settings:
- Background Color: Choose a color using the color picker or enter a hex code.
- Text Color: Choose a color using the color picker or enter a hex code.
- Text Font: Select a font from the available fonts in WordPress.
- Font Size: Set the font size and unit (px, em, rem, etc.).
- Height: Set the height of the topbar and its unit.
- Content Text: Enter the text you want to display in the topbar.
- Custom CSS: Add any additional CSS rules to further customize the topbar.

= How do I add custom CSS? =

In the "Custom CSS" field, you can enter your own CSS rules to style the topbar. For example, use selectors like `.custom-topbar` to target the topbar.
